# Ultra tiny docker image that serves file (less than 600K)

    docker pull superng6/ariang

| Architecture | Tag            |

## How to run Docker Aria2 WebUI AriaNg
--------------------

    docker run --rm -it  -p 80:80/tcp superng6/ariang:latest
